**Tech Stack:** `MERN Stack` · MongoDB · Express · React · Redux Toolkit · Node.js

Financial management platform for **SMEs** — expense tracking, manual journals, and bank transactions.

- Worked on a **financial management platform** for SMEs to manage expenses, journals, and bank transactions
- Integrated **OCR** via **Google Document AI API** for automated document data extraction
- Developed responsive UI with **React** and used **Redux Toolkit** for state management.
- Optimized **Node.js** API endpoints and ensured seamless frontend–backend integration

---

### ⚽ PlayFootball App
**Tech Stack:** `React` · `React Native` · `Strapi` · `PostgreSQL`

Cross-platform app for **football ground bookings**, logistics, referee services, and wallet-based payments.

- **Led full development** of a cross-platform mobile app for ground bookings, logistics, and referee services
- Integrated **Strapi** as a headless CMS for backend data management
- Implemented **custom payment gateways** and optimized API performance
- Revamped UI for better UX and performance; aligned features with client business goals

---

### 🅿️ ParkDNA
**Tech Stack:** `React` · `TypeScript` · `React Query`

US-based **parking discovery & reservation** platform.

- Worked as **frontend developer** on a platform for discovering and reserving parking spaces
- Implemented **React Query** for performance and API consistency
- Collaborated with **cross-functional US teams** on UI design and feature delivery

---

### 🗺️ Orixa Web App
**Tech Stack:** `React` · `Firebase` · `Mapbox GL`

Geospatial platform for **Areas of Interest (AOI)** visualization on interactive maps.

- Developed a **geospatial visualization platform** from scratch to process AOIs and render them on maps
- Integrated **Firebase Auth** for secure authentication
- Used **Mapbox GL** for real-time geospatial rendering and map optimization
- Built **modular, reusable components** and integrated **REST APIs**
